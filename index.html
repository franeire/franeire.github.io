Projects

Some of my projects

Automation

Process Automation

Part of my role is to identify processes within our business that could be automated. To achieve this there are several steps involved.

  1. Document every step of the process, including screenshots
  2. Create a process map, including all decision points
  3. Design the process on BluePrism using these documents
  4. Test process
  5. Put process into production environment.

Create Dashboards

Creating Dashboards

In my previous role on reporting, I had to design a dashboard using Microsoft Excel, based on our total cases.

  1. Generate the report used for the dashboard
  2. Using the attribute fields, identify the filters to apply to the dashboard
  3. Program the macro to build the dashboard
  4. Test the macro

Case management analysis

Case Prediction Model

For my 4th year project, I am currently working on building a prediction model for my Departments case management system. It will try to identify surges in case numbers by individual area, and recommend staff numbers per team based on workload

  1. Generate the all case details report
  2. Preprocess the data, to remove invalid data and outliers
  3. Train the model to identify areas with large workloads and at what time of the year
  4. Based on staff level report, recommend number of staff per team based on likely workload

Fantasy Football Predictor

Fantasy Football Predictor

For my 3rd year project, I designed a basic web app that could make player predictions for fantasy football, based on the player and teams present stats.

  1. Identify the areas you want the user to be able to filter by
  2. Design the application based on these requirements
  3. Build and test the sql database containing the player and team data
  4. Build the sql queries, and allow the user to filter the results
  5. Compare two or more players and visually identify the most likely points return
